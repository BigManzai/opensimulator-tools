README

Another very basic script to generate a bunch of set user level lines for pasting
into an OpenSimulator ROBUST console.  One use is to diable bots used for load testing.


Ein weiteres sehr einfaches Skript, mit dem Sie eine Reihe von benutzerdefinierten Zeilen für das Einfügen erstellen können
in eine OpenSimulator ROBUST Konsole. Eine Verwendung besteht darin, Bots zu dioden, die für Belastungstests verwendet werden.
