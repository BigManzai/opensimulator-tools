README

Another very basic script to generate a bunch of reset user password lines for pasting
Into an OpenSimulator ROBUST console.  One use is to reset bot scripts


Ein weiteres sehr einfaches Skript, um eine Reihe von Reset-Benutzerpasswortzeilen zum Einfügen zu generieren
In eine ROBUST-Konsole von OpenSimulator. Eine Möglichkeit besteht darin, Bot-Skripte zurückzusetzen
