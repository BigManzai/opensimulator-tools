A very basic script to generate a bunch of create user lines for pasting into an OpenSimulator ROBUST
Console in order to generate many users simultaneously (e.g. for bot testing purposes


Ein sehr einfaches Skript, um eine Reihe von Benutzerzeilen zum Einfügen in einen OpenSimulator ROBUST zu generieren
Konsole, um viele Benutzer gleichzeitig zu generieren (z. B. für Bot-Testzwecke)
