README

A very basic script to analyze an xml package that can be imported via a Singularity or Imprudence viewer's import xml facility.
From reports, the import package used by Firestorm is slightly different so this may not work on that.
Currently just checks whether required textures are present
XML package currently needs to be unzipped first

Ein sehr einfaches Skript zum Analysieren eines XML-Pakets, das über die Import-XML-Funktion eines Singularity- oder Imprudence-Viewers importiert werden kann.
Aus Berichten ist das von Firestorm verwendete Importpaket etwas anders, daher funktioniert das möglicherweise nicht.
Prüft gerade, ob benötigte Texturen vorhanden sind
Das XML-Paket muss derzeit zuerst entpackt werden
