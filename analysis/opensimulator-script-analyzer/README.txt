README

This is a a python script to analyze one or more LSL
files (as extracted from an OAR, for instance) and list which LSL/OSSL/mod, etc. functions
are being used.

Probably the biggest immediate addition is to say which files these are actually found
in if one needs to delve deeper.


Dies ist ein Python-Skript, um eine oder mehrere LSL zu analysieren
Dateien (z. B. aus einem OAR extrahiert) und listen auf, welche LSL/OSSL/mod usw. funktionieren
werden benutzt.

Wahrscheinlich ist die größte unmittelbare Ergänzung zu sagen, welche Dateien diese tatsächlich gefunden werden
in wenn man tiefer eintauchen muss.
